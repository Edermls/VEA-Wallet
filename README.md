# 💰 V.E.A Wallet

Simulador de carteira digital desenvolvido em **Java (Spring Boot)** com interface web moderna.
O projeto tem como objetivo ensinar conceitos de **educação financeira**, como investimentos, empréstimos e controle de saldo.

---

## 🚀 Funcionalidades

* 💵 Adicionar saldo
* 💸 Retirar saldo
* 📜 Histórico de transações
* 📊 Simulação de investimento (juros compostos)
* 💰 Investimento real (com retorno automático)
* 💳 Simulação de empréstimo
* 📉 Controle de dívida acumulada
* 💸 Pagamento de dívida

---

## 🧠 Conceitos aplicados

* Juros compostos
* Educação financeira
* API REST
* Integração Front-end + Back-end
* Manipulação de estado (saldo e dívida)

## ▶️ Como executar o projeto

### 🔹 1. Clonar o repositório

```bash
git clone https://github.com/SEU-USUARIO/SEU-REPO.git
cd SEU-REPO
```

---

### 🔹 2. Executar o projeto

```bash
mvn spring-boot:run
```

ou

```bash
java -jar target/*.jar
```

---

### 🔹 3. Acessar no navegador

```
http://localhost:8080
```
